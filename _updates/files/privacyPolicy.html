Paradigm Privacy Policy

The Paradigm Team believes privacy is a human right, and our products are developed with this in mind.

Data Collection

Paradigm collects as little data as possible to provide the best service and experience we can offer for our customers. All processing is done on the device, so we don't need any personally identifying information.

Device Language Code Personalization Translate server-side data to your language, if needed No No Automatically

Device OS Name, Version, and App Version Analytics Make business decisions based on user-base Yes No Automatically
IP Address Security Track, validate, process, and handle malicious requests Only if flagged No Upon invalid request or not passing sever-side security checks.
